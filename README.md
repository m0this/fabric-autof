# AutoF - Automatic Death Reactions

A small, client-side Fabric mod that automatically sends a message when a player dies.

![AutoF Icon](src/main/resources/assets/deathrespect/icon.png)

## Features
- Sends a customizable chat message on player death
- `%player%` placeholder support
- Optional random or fixed delay (0–10 seconds)

## Requirements
- Fabric API
- (Optional) ModMenu

## Configuration
- Via ModMenu:
Mods → AutoF → Configure

- Or manually:
config/deathrespect.json
